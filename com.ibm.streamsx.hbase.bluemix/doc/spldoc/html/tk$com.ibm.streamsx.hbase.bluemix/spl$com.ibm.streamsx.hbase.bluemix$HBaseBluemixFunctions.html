rstring hbaseDeleteTable(rstring baseUrl, rstring username, rstring password, rstring tableName)

Delete an HBase table.

Parameters

  • baseUrl: URL used to access HBase, should be http://.../hbase
  • username: Username if access is via the Knox gateway. May be empty.
  • password: Password for the Knox gateway. Maybe be empty.
  • tableName: Name of the table to create.

Returns

  • the result of the HttpDelete request. May contain an exception in the case of an error and will be empty otherwise.

void hbaseMakeTable(rstring baseUrl, rstring username, rstring password, rstring tableName, list<rstring> columnFamilies)

Create an HBase table.

Parameters

  • baseUrl: URL used to access HBase
  • username: Username if access is via the Knox gateway.
  • password: Password if access is via the Knox gateway.
  • tableName: tableName to be created.
  • columnFamilies: List of column families to put in the table.

rstring hbasePut(rstring baseUrl, rstring username, rstring password, rstring tableName, rstring row, rstring columnFamily, rstring columnQualifier, rstring value)

Put a single entry into an HBase table.

Parameters

  • baseUrl: URL used to access HBase, eg https://host:port/gateway/default/hbase
  • username: username for Knox gateway, if needed.
  • password: password for the Knox gateway, if needed.
  • tableName
  • row: Row for entry to be added.
  • columnFamily: Column family of entry to added.
  • columnQualifier: Column qualifier of entry to be added.
  • value: Value of entry to be added.

Returns

  • A string describing the result of the put request. Will be empty on success and may contain an exception otherwise.

RowType hbaseGet(rstring baseUrl, rstring username, rstring password, rstring tableName, rstring row)

Get all the entries in the given row.

Parameters

  • baseUrl: URL to use https://ipaddr:port/gateway/default/hbase
  • username: Username for Knox gateway, if needed.
  • password: Password for Knox gateway, if needed.
  • tableName: Name of the table
  • row: row to get.

Returns

  • Tuple containing the row and all values in that row.
