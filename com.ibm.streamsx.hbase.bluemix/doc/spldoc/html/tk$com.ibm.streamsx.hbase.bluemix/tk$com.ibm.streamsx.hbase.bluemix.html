General Information

The HBase Bluemix toolkit is used for interacting with HBase via its REST API. As the name implies, the toolkit is best suited for interacting with HBase deployed on Bluemix.

The toolkit contains spl functions for interacting with HBase. You can use these SPL functions in Custom operators and Functors as needed in your application. This requires version 2.7 or better of the com.ibm.streamsx.inet toolkit, as well as version 1.0 or better of the com.ibm.streamsx.bytes toolkit.

This functionality is provided in the com.ibm.streamsx.hbase.bluemix namespace.

Developing and running applications that use the HBase Bluemix Toolkit
To create applications that use the HBase Toolkit you must configure either Streams Studio or the SPL compiler to be aware of the location of the toolkit.
