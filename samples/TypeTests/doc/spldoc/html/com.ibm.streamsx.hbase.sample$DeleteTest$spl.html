composite DeleteTest

Tests the delete operation with a variety of types (not a basic sample).

The composite takes the RowType, ColumnQualifierType, and ValueType as arguments. (the columnFamily is assumed to be rstring--it's difficult to create tables that use non-rstring values as columnFamily names, so that is not well-tested).

It also takes a getter function for the row, columnqualifier, and value.

Then, it produces tuples with those types, puts the tuples in HBASE, and then deletes them using a conditional delete that will only succeed if the the tuple is there.
