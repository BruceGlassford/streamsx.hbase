com.ibm.streamsx.hbase::HBASEScan

com.ibm.streamsx.hbase -> com.ibm.streamsx.hbase-> HBASEScan

Scan an HBASE table. Each row/columnFamily/columnQualifer/value entry is mapped to a tuple, with the row populating the row attribute, the columnFamily populating teh columnFamily attribute, the columnQualifier attribute populating the columnQualifier attribute, and the value populating the value attribute. The value may either be a long or a string, all other values must be rstring. An optional start row and end row may be specified.
